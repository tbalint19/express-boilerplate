# ExpressJS boilerplate app

## Features
- Sequlize ORM
- Prometheus middleware (metrics)
- Morgan JSON logger
- Google OpenID authentication
- Auth models (user, role, permission)
- Unit & Integration tests (with jest)
- Axios http client
- Prettier format
- CI/CD

## Requirements
- node
- npm
- git
- docker
